Simple OpenGL demonstrations:

4.1 - OpenGL 3.2 and 4.1 related demos. Primary dev environment is OS X Lion
|
+- triangle - A spike using glfw-2.7.2. Loads shaders, draws a triangle.
|
+- cube - builds on the triangle demo by adding a uniform variable
|
+- qt - demonstrates getting an OpenGL 3.2 context using qt 4.7 under OS X Lion.
|
+- trackball - demonstrates a virtual trackball.
|
+- objreader - a hacked version of Nate Robbin's OBJ reader for OpenGL 3.2
|
+- texture - a simple texture mapping example
|
+- fbo - demonstrate render-to-texture using a framebuffer object
